# LaneShooter

A Raylib C semi-3D pixel art shooter game

## Requirements
- [Raylib 5.0](https://www.raylib.com/)
- C compiler (GCC, Clang, or MSVC)

## Build (Linux)
```bash
make
./build/LaneShooter
